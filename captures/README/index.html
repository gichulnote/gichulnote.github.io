캡쳐용 페이지 가이드

OG 이미지와 스플래시 화면 생성을 위한 캡쳐용 페이지입니다.

페이지 목록

파일용도크기출력
og-image.astro웹사이트 SNS 공유용1200x630public/og-image.png
splash.astro앱 스플래시 화면1080x1920public/splash.png

캡쳐 방법

1. 빌드 후 Preview 서버 실행

cd website
pnpm build && pnpm preview

2. Playwright로 캡쳐

# OG Image (1200x630)
npx playwright screenshot \
  --viewport-size="1200,630" \
  --full-page \
  "http://localhost:4321/captures/og-image/" \
  "public/og-image.png"

# Splash (1080x1920)
npx playwright screenshot \
  --viewport-size="1080,1920" \
  --full-page \
  "http://localhost:4321/captures/splash/" \
  "public/splash.png"

주의: pnpm dev가 아닌 pnpm preview에서 캡쳐해야 Astro 디버그 툴바가 포함되지 않습니다.

로고 변경 방법

현재 로고는 CSS로 만든 “G” 텍스트입니다.

현재 구조

<!-- og-image.astro, splash.astro 공통 -->
<div class="logo">G</div>
<div class="app-logo">G</div>  <!-- 목업 내부 -->

이미지 로고로 변경 시

  1. 로고 이미지를 public/logo.png에 저장

  2. HTML 변경:

<!-- 메인 로고 -->
<img src="/logo.png" class="logo" alt="기출노트" />

<!-- 목업 내부 로고 (작은 사이즈) -->
<img src="/logo.png" class="app-logo" alt="G" />
  1. CSS 수정 (필요시):
.logo {
  width: 64px;
  height: 64px;
  /* background 관련 속성 제거 */
  object-fit: contain;
}

.app-logo {
  width: 28px;  /* og-image */ 또는 36px /* splash */
  height: 28px;
  object-fit: contain;
}

수정 필요 파일

  • src/pages/captures/og-image.astro - 2곳 (메인 로고 없음, 목업 내부만)
  • src/pages/captures/splash.astro - 2곳 (메인 로고, 목업 내부)
